Team Pulse v0.15.3

What this is
- A single-file browser app for tracking direct reports, cadence touchpoints, PDCs, CV reviews, notes, hire dates, and promotion dates.
- The durable browser release writes its files into a user-chosen Team Pulse folder.

How to use
1. Open team-pulse-v0.15.3.html in Edge or Chrome.
2. Click Choose folder and select the folder where Team Pulse should keep its files.
3. Use the Add direct report button inside the Direct reports section to open a blank workspace.
4. Click any direct report row to open that person’s workspace.
5. Update profile fields, development focus, and notes, then click Save changes.
6. Use the Log meeting button in the workspace to open the meeting modal above the direct report workspace.
7. In the meeting modal, either:
   - log the meeting manually, or
   - click Import .ics to import a calendar file into that direct report.
8. Review or edit existing meetings from the Meeting history section.

Durability behavior
- Team Pulse keeps the JSON file authoritative.
- Autosave remains the primary persistence path and queues after data-changing actions with a short debounce of about 160 ms once a Team Pulse folder is connected.
- Clicking Save now writes immediately and the save-state pill shows the manual save time.
- The connected folder receives:
  - team-pulse.json
  - team-pulse.backup.json
  - team-pulse.daily.json
  - team-pulse.monthly.json
  - SCHEMA.md
  - quarterly plain-format export files when run
- IndexedDB is only used to remember the chosen folder handle in the same browser profile.

Notification behavior
- Transient toast banners dismiss after about 2 seconds.

Container behavior
- Containers do not close when clicking outside them.
- Use the Close, Cancel, or Save buttons to close a container.
